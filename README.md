# microcrawler-client
Microcrawler Client

## Getting started

***Clone Sources***

```
git clone https://github.com/AplloCrawler/microcrawler-client.git
```

***Enter Directory with Sources***

```
cd microcrawler-client
```

***Install Dependencies***

```
npm install
```

***Run Microcrawler Client***

```
bin/microcrawler-client.js
```
